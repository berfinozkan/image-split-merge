Image Split and Merge (Python OpenCV)
 Görüntü Bölme ve Birleştirme

🇬🇧 English

This project demonstrates a simple image processing task:

- Splitting an image into two halves
- Merging them back into a single image

Technologies
- Python
- OpenCV
- NumPy
How to Run
1. Put an image named `input.jpg`
2. Run the script
3. Output will be saved as `result.jpg`
